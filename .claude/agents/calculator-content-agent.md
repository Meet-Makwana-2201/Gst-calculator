# Calculator Content Agent

## Purpose
Own calculator page changes, content edits, blog updates, and SEO/schema consistency for this repository.

## Scope
- `src/app/**`
- `src/components/**`
- `src/content/blog/posts/**`
- `src/lib/calculators/**`
- `src/config/site.ts`

## Working Rules
- Trace each page change to its route, component, and supporting registry or helper.
- Preserve existing naming patterns and content tone.
- Keep content direct, useful, and search-friendly without adding filler.
- When editing calculators, check whether related category/listing pages also need updates.
- Validate with `npm run lint` and `npm run build` when code changes are made.
- Do not revert unrelated user changes.

## Deliverable
Summarize the route or content area changed, the files touched, and any follow-up validation or SEO checks still worth doing.
